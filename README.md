# 🌐 MySite

Личный каталог — игры, фильмы и рецепты в одном месте. Чистый HTML/CSS/JS, без фреймворков.

---

## 📁 Структура проекта

```
MYSITE/
│
├── index.html                  # Главная страница (навигация)
│
├── pages/                      # Страницы разделов
│   ├── games.html              # Каталог игр
│   ├── movies.html             # Каталог фильмов
│   └── recipes.html            # Каталог рецептов
│
├── assets/
│   ├── css/
│   │   ├── style.css           # Основные стили (десктоп)
│   │   └── style-mobile.css    # Адаптивные стили (≤ 1100px)
│   ├── js/
│   │   └── js.js               # Логика: фильтры, поиск, статусы
│   └── images/                 # Обложки игр, фильмов, рецептов
│
└── README.md
```

---

## 🚀 Запуск

Просто открой `index.html` в браузере — никаких серверов и зависимостей не нужно.

Или используй расширение **Live Server** в VS Code для автообновления при редактировании.

## 📄 Страницы

### 🎮 Игры (`pages/games.html`)

Каталог игр с фильтрами по режиму (Single/Multi), производительности (Тянет/Не тянет), сети (Radmin/NoRadmin) и статусу прохождения.

### 🎬 Фильмы (`pages/movies.html`)

Список фильмов с фильтрами по жанру (Хоррор, Фантастика, Драма), качеству (4K, Full HD) и статусу просмотра.

### 🍳 Рецепты (`pages/recipes.html`)

Рецепты с фильтрами по категории (Завтрак, Обед, Ужин, Десерт, Перекус), сложности и статусу «Готовил / Не готовил». Время приготовления показано в бейдже на обложке.

## ➕ Как добавить карточку

Скопируй любую `.game-card` на нужной странице и замени:

```html
<div class="game-card" data-tags="ТЕГИ ЧЕРЕЗ ПРОБЕЛ">
  <div class="card-image">
    <img src="../assets/images/ФАЙЛ.jpg" alt="НАЗВАНИЕ" />
    <span class="badge-year">ГОД</span>
  </div>
  <div class="card-content">
    <h3>НАЗВАНИЕ</h3>
    <p>ОПИСАНИЕ (кратко, 1–2 предложения)</p>
    <div class="tags">
      <span class="tag tag-КЛАСС">ТЕКСТ</span>
      <!-- статусный тег — кликабельный -->
      <span class="tag tag-undone status-toggle" data-state="undone"
        >Не пройдено</span
      >
    </div>
  </div>
</div>
```

**Доступные теги по страницам:**

| Страница | Теги                                                                                     |
| -------- | ---------------------------------------------------------------------------------------- |
| Игры     | `single` `multi` `runs` `noruns` `radmin` `noradmin` `done` `undone`                     |
| Фильмы   | `horror` `sci-fi` `drama` `4k` `fhd` `done` `undone`                                     |
| Рецепты  | `breakfast` `lunch` `dinner` `dessert` `snack` `easy` `medium` `hard` `tried` `nottried` |

## 🎨 Кастомизация

Все цвета и параметры находятся в переменных в начале `assets/css/style.css`:

```css
:root {
  --accent: #8a4fff; /* основной акцентный цвет */
  --pink: #ff4b9e; /* второй акцент (градиенты) */
  --bg: #07080c; /* цвет фона */
  /* ... */
}
```

Мобильные стили редактируй в `assets/css/style-mobile.css` — там только `@media` блоки.
